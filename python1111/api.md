# str型

# メソッド
* upper  
    * 大文字にする
    ```pytohn
    msg = "osaka-is-good"
    print(msg.upper())
    ```
* lower 
    * 小文字にする
    ```pytohn
    msg = "osaka-is-good"
    print(msg.lower())
    ```

* replace
    * 置換
    ```pytohn
    msg = "osaka-is-good"
    print(msg.replace("osaka", "japan"))
    ``` 


# list型
* insert
    * 要素を指定した位置に追加する
    ```python
    my_list = ["A", "B", "C"]
    my_list.insert(2, "C")
    ```
